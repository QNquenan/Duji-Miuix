# 嘟迹

一款装置管理应用，记录每一件装置的价格、购买日期与日均花费。

嘟迹是一款面向个人使用的本地化 Android 应用，用于管理日常物品、记录重要日子，并通过桌面小部件快速查看信息。

## 功能

### 我的物品

- 记录物品名称、图标、价格、购买日期和备注。
- 支持列表视图与网格视图切换。
- 展示物品平均日均花费、总价值和物品数量统计。
- 支持搜索、排序、置顶、编辑、删除和详情查看。

### 那些日子

- 记录倒数日、纪念日和生日。
- 支持公历、农历日期，以及每周、每月和每年重复周期。
- 支持表情、自定义图标、搜索、排序、列表视图和网格视图。
- 在详情页查看日期、类型、当前状态和备注。

### 桌面小部件

- 我的物品（宽版）：显示物品总价值、物品数量和日均花费。
- 我的物品（方形）：显示指定物品的图标、名称、日期和价格信息。
- 那些日子：显示指定日子的名称、倒数数字、单位和日期。
- 添加小部件时可以选择对应的数据，数据变化后会自动同步到桌面。
- 点击小部件可以直接进入应用中的对应页面或详情抽屉。

### 数据与设置

- 使用本地 DataStore 保存物品和日子数据。
- 支持将数据导出为 JSON 文件，也支持从 JSON 文件合并导入。
- 支持浅色、深色和跟随系统的外观模式。
- 提供应用内更新日志页面和可选的预测性返回手势。

## 技术栈

- Kotlin
- Jetpack Compose
- Miuix
- Jetpack DataStore Preferences
- Android AppWidget `RemoteViews`
- `minSdk 33`，`targetSdk 36`，`compileSdk 37`

## 构建

在 Windows 环境下执行：

```powershell
.\gradlew.bat :app:assembleDebug --console=plain
```

生成的 Debug APK 位于：

```text
app/build/outputs/apk/debug/app-debug.apk
```
